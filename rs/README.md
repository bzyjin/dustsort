## Interface

| Function      | Usage                                             					   |
|---------------|--------------------------------------------------------------------------|
| `sort`        | Sort [`Ord`](https://doc.rust-lang.org/core/cmp/trait.Ord.html) elements |
| `sort_by`     | Sort elements with a comparison function           					   |
| `sort_by_key` | Sort elements with a mapping from elements to keys			   		   |
